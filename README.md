# ReShap: Interpretable Reinforcement Learning for High-Dimensional Multi-Action Settings

## Install
1. Clone this repository
   ```bash
   git clone https://github.com/qyy752457002/Interpret-DRL-using-SHAP-Project.git
   ```
2. Install dependencies (Please use Python 3.10.11)
   ```bash
   pip install -r requirements.txt
   ```

## How to Run
1. MsPacman experiments: MsPacman A2C.ipynb, MsPacman DQN.ipynb, MsPacman PPO.ipynb

2. RoadRunner experiments: RoadRunner A2C.ipynb, RoadRunner DQN.ipynb, RoadRunner PPO.ipynb

3. CartPole experiments: CartPole A2C.ipynb, CartPole DQN.ipynb, CartPole PPO.ipynb

4. Human study: Analysis.ipynb under Human Study directory

5. Fidelity analysis: Fidelity Plot.ipynb
